BIM AUTOMATION PORTFOLIO

建築BIMの実務を、コードで自動化する。

Rhino / Grasshopper / Python で自作した 45+ の建築BIM自動化ツールのカタログです。3Dモデルから製作図を自動作図する Make2D パイプライン、IFC 相互運用ツール群、ローカルLLM を組み込んだ GHノード自動生成基盤まで——要件定義・設計・実装・チーム展開のすべてを一人で担いました。各ツールはアニメーションデモ付きで「何ができるか」が一目で分かります。

製作：中村 勇太　／　このサイト自体も AI（Claude）をディレクションして構築 — コードベースの調査・解説の執筆・アニメデモの生成・サイト実装までを AI エージェントに伴走させて仕上げています。

全体マップ

6つのカテゴリと、その間の主なデータの流れです。ブロックをクリックすると、その分類の一覧へジャンプします。IFCと鉄骨モデリングが「図面化パイプライン」へ素材を送り、AI基盤がノード自体を自動生成、運用系が全体を横断で見張る——という関係になっています。

Make2D 図面化パイプライン

このカタログの中心。3Dモデルと2D図面から情報を集め → 切断を計画 → Make2Dで断面を起こし → 元の図面ファイルへ書き戻すという一連の流れを、たくさんのGHノードが分担しています。全体像は詳細図でどうぞ。
