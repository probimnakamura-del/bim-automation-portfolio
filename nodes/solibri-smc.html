Solibri／SMC 抽出ツール群

Solibri の .smc から中身の IFC を取り出し、いらない属性をそぎ落として軽量化——フォルダに置くだけで自動処理。

稼働中 Python スクリプト群（Tk GUI） 【】付き .bat ランチャー起動

MOTION PREVIEW

動作デモ

PREVIEW⟲ LOOP 10s

SMC から中身の IFC を抜き出し → 監視フォルダで自動検出 → 属性を除去して軽量化、という一連の流れを10秒でループ再生しています。

1なにをするツール？

Solibri（BIMの検図ソフト）が扱う .smc ファイルから、中身の IFC モデルを取り出すための小さな道具箱です。.smc は実は ZIP なので、その中に入っている IFC を安全に抜き出します。

さらに、取り出した IFC から プロパティ（Pset）・材質・数量・分類といった属性情報をごっそり除去して、形状だけの軽い IFC を作ることもできます。属性が多い巨大な IFC は Rhino や Solibri で開くと重いので、「形だけ確認したい」「後工程に渡す前に軽くしたい」ときに効きます。

そして フォルダを見張って、新しい IFC が置かれたら自動で軽量化まで実行する監視モードも用意されています。Solibri から書き出す先のフォルダを指定しておけば、手を動かさずに軽量 IFC が溜まっていきます。

※ SMC＝Solibri Model Checker のモデル保存形式（中身は ZIP）。／ IFC＝BIM モデルを各社ソフト間でやり取りするための共通ファイル形式。／ Pset（プロパティセット）＝各部材にぶら下がる属性のまとまり。

2できること（操作 → 結果）

GHノードではなく、単体で動く Python ツールの集まりです。役割ごとに4本のスクリプトに分かれています。

ツール構成
操作 / スクリプト結果
solibri_pipeline.py全部入りの司令塔。SMC選択 → Solibriで書き出し → 監視フォルダ検出 → プロパティ除去まで、一連を1画面で回す。

3使い方

  1. ツールを起動するC:\Python1 の 【】付き .bat ランチャー、または py solibri_pipeline.py で起動する。
  2. SMC を選ぶ対象の .smc を選択。中に入っている IFC の候補が一覧表示される。
  3. Solibri から IFC を書き出すSolibri のモデルパネルで対象を右クリック → エクスポート → IFC。書き出し先を監視フォルダに設定しておく。
  4. 自動検出させる監視フォルダを指定して監視開始。新しい IFC が置かれると自動で検出し、プロパティ除去 ON なら軽量化まで走る。
  5. 軽量 IFC を受け取る出力フォルダに 形状だけの軽い IFC が生成される。削除件数のレポートで何を落としたか確認できる。
  6. GH へ渡すできた IFC を Tekla/Solibri IFC→GHインポータ で Grasshopper に取り込む。

4つながり（ワークフロー）

Solibri／SMC 抽出ツール群（このツール） ↓
後工程 Tekla/Solibri IFC→GHインポータ

SMC からの抽出・軽量化は、IFC 連携の入口。ここで整えた IFC を、Grasshopper 取り込みや Rhino 変換の各ツールへ流します。

5実データでの手順

📦 手元の .smc ／ .ifc で試す

専用サンプルはありません。実際のプロジェクトの .smc（または重い IFC）で試すのが一番わかりやすいツールです。

  1. 重い IFC を1本用意属性がたっぷり入った実案件の IFC を1つ選ぶ。ファイルサイズを覚えておく。
  2. 軽量化だけ単体で試すpy ifc_strip_props.py model.ifc -v を実行。削除件数のログと、生成された軽量 IFC のサイズを見比べる。
  3. 監視モードを体験空フォルダを監視対象に指定 → そこへ IFC をコピー。数秒で自動検出され、軽量化まで走ることを確認する。

6注意点・既知の罠

  • ⚠ プロパティ除去は「戻せない」。元 IFC は必ず残す

    軽量化すると Pset・材質・数量・分類がすべて消えます。後で属性が必要になっても復元できません。属性を使う工程（拾い出しや検索）には除去前の IFC を渡し、軽量版は形状確認・受け渡し専用と割り切ってください。

  • ⚠ 監視は「起動後に置かれたファイル」だけが対象

    フォルダ監視は起動時点にあったファイルを既存扱いにして無視します。すでに入っている IFC は処理されないので、監視開始後に置き直すか、単体スクリプトで手動処理してください。書き込み途中を掴まないよう、サイズが安定するまで待つ仕組みが入っています。

  • ⚠ SMC の中に複数モデルが入っていることがある

    .smc は ZIP なので、中に 複数の IFC が同梱されている場合があります。取り出し時にどのモデルが目的のものか、内部エントリ名で確認してください。

  • ⚠ 起動は【】付きの .bat ランチャーから

    この一式は C:\Python1 直下の 【】付き .bat ランチャーから呼ばれる運用です。単体で py 実行する場合は ifcopenshell が入った環境が必要です。
