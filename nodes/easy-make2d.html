図面化パイプライン (Make2D)

簡単Make2D easy_make2d

Grasshopper要らず。切りたい場所をクリックするだけで、その場に断面図が出てくる。

稼働中 v3.1 相当 Rhinoコマンド（GH不要）

MOTION PREVIEW

動作デモ

PREVIEW⟲ LOOP 10s

モデルを選ぶ → 切りたい場所をクリック（実断面線をライブプレビュー）→ その場で断面図が原点付近に1:1配置、という流れを10秒でループ再生しています。

1なにをするツール？

Grasshopperを一切開かず、Rhinoのコマンドラインから起動して「切断位置をクリックするだけ」で、その場に2Dの断面図・平面図をつくる手軽版のMake2Dです。対象を選び、切りたい場所をクリックすると、原点付近の空いた場所へ1:1スケールで断面線がきれいに並びます。

いちばんの特長は、カーソルを動かすと切断面に現れる「実際の断面線」がその場でプレビューされる点。「切ってみないと分からない」がありません。奥行き（矢視先を何mmまで描くか）も立体プレビューされます。大きな図面化パイプライン（GH_Make2DExecutor）の解決済みコアを1系統に絞って移植した、"すぐ1枚欲しい" ときの相棒です。

※ Make2D＝3Dモデルを指定した視点で2Dの線画（平面図・断面図など）に落とす処理。／ 矢視＝断面をどちら側から見るかの向き。

3使い方

  1. 起動するRhinoで 断面 または M2D と打つ（EasyM2D.rui のアイコンでも可）。GHは開かなくてOK。
  2. 対象を選ぶ先に選んでおけば即採用。クリックで追加、空所からドラッグで窓選択。ワークセッション参照モデル・ブロックも選べる。
  3. モードを決める水平・部材直交・外観など。勾配部材は 「2点指定」 で切断線に沿わせると、紙面でも実長が読めます。
  4. 切断位置をクリックカーソル追従の断面線プレビューを見ながら位置決め → クリックで確定。すぐ原点付近に断面図が現れます。
  5. 続けて作る／取り消すクリックを繰り返して連続作成。間違えたら 取消 で直前を丸ごと削除（符号も再利用）。
  6. 仕上げる設定 で断面塗り・尺度（1:N）・隠れ線などを調整。Enter / Esc で終了。

4つながり（ワークフロー）

単独で使えます。 GHやパイプラインの配線なしで、Rhinoから直接1枚ずつ断面図を起こせます。

きちんと計画して量産・図面へ書き戻したいときは、GH版の GH_Make2DExecutor と Make2Dパイプライン をどうぞ。本ツールはその「クイック単発版」にあたります。

5サンプルモデルでデモ

🧩 サンプルモデル\01_柱梁フレーム.3dm

2×1スパンの柱梁フレーム＋通り芯線＋ガセット／浮きピース入り。切断位置クリックで断面図がすぐ出るのを体験できます。

  1. モデルを開くサンプルモデル\01_柱梁フレーム.3dm を開く。
  2. 起動して全選択断面 と打ち、フレーム一式を選ぶ（先に選んでから起動でもOK）。
  3. 水平切断を試すモードを「水平切断（上から）」にして、柱の中ほどをクリック → 床伏図が原点付近に1:1で出る。
  4. 2点指定も試す「2点指定」で通り芯 X2 に沿って2点を打ち、3点目で矢視側を決める → 断面図が並ぶ。取消 で気軽にやり直せます。

6注意点・既知の罠

  • ⚠ 重いモデルではライブプレビューが自動OFF

    断面線プレビューは対象が多すぎる（約600ジオメトリ超）と自動でオフになります。プレビューが出なくてもクリックすれば断面図は作れます。動作が重いと感じたらそのまま進めてOK。

  • ⚠ 水平切断だけ線が出ないときはZ基準（1FL）を確認

    水平断面だけ線がゼロになる場合、切断高さが階の基準（1FL）からずれているのが典型。切る高さを部材が通る位置に合わせ直してください（Make2Dパイプライン共通の罠）。

  • ⚠ 断面線が0本のときはビューを1つ開いてから

    内部で既存ビューポートを複製して投影します。全ビューが閉じた特殊な状態だと可視0本になり得るため、通常の作業ビューを開いた状態で実行してください。

  • ⚠ Rhino 8 / RunPythonScript 前提

    IronPython2.7・CPython3の両対応コードですが、動作確認はRhino 8想定です。エイリアス「断面／M2D」は install_m2d_alias.py で登録すると便利です。
