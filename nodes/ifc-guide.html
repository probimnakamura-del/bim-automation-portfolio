どれを使う？（早見チャート）

IFC 関連ツールは目的ごとに役割が分かれています。やりたいことを左の起点から選び、対応するツールのページへどうぞ。結合系はIFCMergerに統合されましたが、単体版も引き続き使えます。

左の「やりたいこと」から、右のツールへ。詳しくは下の早見表のリンクから各ページへ移動できます。

2結合系の使い分け

「複数 IFC の結合」は、いまは統合版 IFCMerger に一本化されています。ただし移植元は無改変で温存されているため、単体版もそのまま使えます。

  • 🧩 まとめて1本化したいなら IFCMerger

    基準＋複数の結合、範囲抽出、HTMLレポートを1つの Web UI で。迷ったらこれ。

  • 🧱 単体版も残っています

    原点キューブでの位置合わせ結合は Join_IFC_model、Tekla色を転写する2ファイル結合は Join_IF_model、巨大IFCの範囲抽出は IFC編集ツール（IFC_Clipper）。
