参照モデルへ着地 GH_DropOnRef

浮いた部材はストンと落下、埋まった部材はせり上げ。参照モデルの上へ0タッチで一括着地。

稼働中 Grasshopper Pythonノード Run立上りで実移動

MOTION PREVIEW

動作デモ

PREVIEW⟲ LOOP 10s

浮いた部材は真下の天端へストンと落下、埋まった部材は部材天端までせり上がり——バラバラなレベルの部材を1ノードで一括0タッチする様子を10秒ループで再生しています。

1なにをするツール？

ワークセッションで参照している下地モデルの上面に、編集中の部材の底面をピタッと突き合わせる（0タッチ）ためのノードです。選んだ部材の一つひとつが、自分の真下にある支持面を自動で選んで個別に着地するので、レベルがバラバラの部材でもまとめて一括処理できます。

賢いのは浮き／埋まりの両対応。宙に浮いている部材は真下の天端へ「落下」し、下地に食い込んで埋まっている部材はその部材の天端まで「せり上げ」て0タッチにします（AllowRaise で引上げのオン／オフ切替）。Gap に数値を入れれば、あえて隙間を残した配置もできます。同じRhinoグループの部材は剛体としてまとめて動くので、梁＋スチフナ＋ガセットといった組立は相対位置を崩さずに着地します。

※ 0タッチ＝隙間ゼロで面を突き合わせること。／ ワークセッション＝別ファイルのモデルを「参照」として重ねて開く仕組み。参照側は動かしません。

2入出力（GHノードの端子）

入力ポート
端子説明
Ids移動する編集部材のGUIDリスト（アクティブモデルの通常オブジェクト。ブロックインスタンス可）。
RefIds任意。着地先に使う参照パーツのGUID。未接続ならワークセッションの全参照を自動スキャン。

端子説明

3使い方

  1. 参照モデルを開く下地になるモデルをワークセッションで参照として読み込んでおく（着地先の天端がある状態）。
  2. 部材を渡す着地させたい編集部材のGUIDを Ids へ接続。着地先を絞りたければ RefIds も指定（未接続なら全参照から自動選定）。
  3. 隙間と挙動を決めるGap で目標隙間（0で0タッチ）、AllowRaise で埋まり時の引上げ可否、GroupAsUnit で組立の一括移動を設定。
  4. プレビューで確認Preview をオンにすると、緑のゴーストと移動量(dz)が画面に出る。想定どおりの天端に着地するか目視。
  5. Runで実行Run を押して離す（立上りエッジ）と実際に移動。Buttonでの運用が安全。ずれても Ctrl+Z で取消可。

4つながり（ワークフロー）

単独で使えます。 参照モデルさえ開いていれば、他のノードにつながず単体で完結する配置補助ツールです。端をライン基準にそろえたいときは、姉妹ツールの GH_AlignEndToLine が同じ操作感で使えます。

5サンプルモデルでデモ

🧩 サンプルモデル\04_落下整列.3dm

梁の上に浮いた部材＋基準ライン。GH_DropOnRef着地とGH_AlignEndToLine整列のデモ用。

  1. モデルを開くサンプルモデル\04_落下整列.3dm を開く（下地の梁は参照、または同一ファイル内でもデモ可）。
  2. 浮いた部材を選ぶ梁の上に浮いている部材のGUIDを Ids に渡す。
  3. プレビューPreview オンで、緑ゴーストが下の梁天端へ落ちる位置を確認。Gap=0 で0タッチ。
  4. Runで着地Run を立上りで押すと実際に落下。Dz でどれだけ落ちたか確認できる。

6注意点・既知の罠

  • ⚠ Runは「立上りエッジ」でのみ実移動。まずPreview

    実際の移動は Run が False→True になった瞬間だけ起きます（Button推奨）。Toggleで放置しても再適用は1回だけ。dzは毎回いまの位置から再計算する冪等設計なので、適用後はdz≒0になります。押す前に必ず Preview の緑ゴーストで着地先を確認してください。

  • ⚠ 参照オブジェクトはワークセッションで読み込んでおく

    参照モデルの列挙には ObjectEnumeratorSettings(ReferenceObjects=True) が必要で、通常の走査では参照が拾えません。着地先が反応しないときは、下地モデルがワークセッション参照として開かれているかを確認してください。

  • ⚠ 天端は軸並行BBoxの最大Z。傾いた面は1点接触

    「上面／底面」は軸に平行なバウンディングボックスの max／min Z で判定します。水平材やスラブのような平らな天端では正確ですが、著しく傾いた天端は1点接触になる点に留意してください。

  • ⚠ 遠い別レベルへは落とさない（SearchRange）

    SearchRange（既定1000mm）の外にある天端には吸着しません。これは別の階などへの誤吸着を防ぐためです。意図的に遠いレベルへ落としたいときは値を広げてください。
