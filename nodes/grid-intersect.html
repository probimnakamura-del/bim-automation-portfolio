図面化パイプライン (Make2D)

通り芯交点抽出 GH_GridIntersect

3Dモデル上の通り芯の交わりを「X1-Y3」の名前付きでひとつずつ拾い上げる。

稼働中 v2.0（設定ポップアップ＋永続化） GH Python ノード

MOTION PREVIEW

動作デモ

PREVIEW⟲ LOOP 10s

3Dモデルの通り芯レイヤーから交点を1つずつ十字マークで検出し、「X1-Y1」のように名前を付けて出力する流れを10秒でループ再生しています。

1なにをするツール？

3Dモデル側にある通り芯どうしの交わり（交点）を、名前付きで拾い出すノードです。「X1」と「Y3」の通り芯が交わる所を X1-Y3、フロアも付ければ 1FL_X1-Y3 という名前で出力します。この名前付き交点が、あとで2D図面と3Dモデルの位置をぴったり合わせるための「目印」になります。

読み取り元は、主にワークセッションで参照している3Dモデルの「フロアごとに仕分けられた通り芯レイヤー」です。Pick ボタンを押すと「モデル → レイヤー → フロア」の3段ポップアップが開き、どのモデルのどのレイヤーを対象にするかを設定できます。設定は gridx_settings.json へコンポーネントごとに保存されるので、Rhinoを再起動しても選び直す必要はありません。GH_ModelPicker・GH_LayerPicker も同じ働きの姉妹ノードです。

※ 通り芯＝建物の基準線（X1, Y1…）。／ ワークセッション＝複数の.3dmを参照として重ねて扱うRhinoの仕組み。他ファイルのモデルを下敷きにできます。

3使い方

  1. 参照モデルを開く通り芯がフロアごとにレイヤー分けされた3Dモデルを、ワークセッションで参照するかRhinoに開く。
  2. Pickで対象を選ぶPick を押して開く3段ポップアップで、モデル → 通り芯レイヤー → フロアを選ぶ。選択は自動で保存される。
  3. 実行して交点を出すRun を押すと、通り芯どうしの交点が計算され Pts に、名前が FullNames に出る。
  4. ハイライトで確認Highlight をオンにすると、交点に名前タグが浮かぶ。狙った交点が拾えているか目視。
  5. プランナーへ渡す名前付き交点を GH_Make2DPlanner へ。図面とモデルの位置合わせの基準として使われる。

5サンプルモデルでデモ

🧩 サンプルモデル\01_柱梁フレーム.3dm

2×1スパンの柱梁フレーム＋通り芯線入り。通り芯交点抽出・接触検出・Make2D・部材記号のデモ用。

  1. モデルを開くサンプルモデル\01_柱梁フレーム.3dm を開く（またはワークセッションで参照）。通り芯線のレイヤーを確認。
  2. Pickで通り芯レイヤーを選ぶ本ノードを置き Pick → モデル → 通り芯レイヤー → フロアを選択。
  3. Runで交点を出すRun を押し、FullNames に X1-Y1 などが並ぶのを確認。
  4. ハイライトで目視Highlight をオンにして、交点に名前タグが正しく乗るかを見る。

6注意点・既知の罠

  • ⚠ 交点判定は2D投影。高さ違いの線でも交わりを拾う

    交点はすべて2Dに投影してから取ります。フロアごとに正しくレイヤー分けされていれば問題ありませんが、別フロアの通り芯を混ぜて対象にすると、投影上は交わって見えて意図しない交点を拾うことがあります。フロアの仕分けを確認してください。

  • ⚠ 設定はコンポーネントIDごとに gridx_settings.json へ保存

    Pickで選んだ設定は gridx_settings.json にコンポーネント単位で保存され、Rhino再起動後も保持されます。裏を返すと、ノードを作り直すと別IDになり設定が引き継がれないことがあります。その場合は Pick からもう一度選び直してください。

  • ⚠ 参照モデルのレイヤーが選択肢に出ないとき

    ワークセッション参照モデルのレイヤーは、状況によって列挙に出てこないことがあります。対象モデルがきちんと参照アタッチされているか、通り芯レイヤーがそのモデル側にあるかを確認してください。
