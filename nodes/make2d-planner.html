図面化パイプライン (Make2D)

Make2Dプランナー GH_Make2DPlanner

図面から読んだ断面と、3Dモデルの通り芯を名前で照合して、どこをどの向きで切るかの計画を組み立てる司令塔。

稼働中 GH Python ノード make2d_system 構築対象

MOTION PREVIEW

動作デモ

PREVIEW⟲ LOOP 10s

図面から読んだ交点名と3Dモデルの通り芯を名前で突き合わせ、各断面の「切る位置・向き・範囲」を計画（Plan_JSON）に落とす流れを10秒でループ再生しています。

1なにをするツール？

Make2D図面化パイプラインの頭脳です。2D図面から抽出した断面データ（統合JSON）を読み込み、3Dモデル側の通り芯交点と「名前」で照合します。たとえば図面に「X1通りで A-A断面」と書いてあれば、モデルの X1 通りの位置を探し当てて、そこに切断面を組み立てます。

断面は鉛直切断（断面図）・水平切断（床伏図・天井伏図）・立面ビューの3種を自動で判別し、それぞれについて「切る面・見る向き（矢視）・切り取る範囲（クリップ）」をモデル空間に用意します。結果は Plan_JSON という計画データにまとめ、そのまま GH_Make2DExecutor へ渡します。3Dモデルの交点を渡さなくても、図面座標モードでJSONの内容だけから計画を出せるのも便利なところです。

※ 統合JSON＝2D図面から通り芯・断面符号・レベル線などを読み取って1本にまとめたデータ（GH_SectionDataJSON が作る）。／ 矢視＝断面をどちら側から見るかの向き。

3使い方

  1. 統合JSONを渡すGH_SectionDataJSON が作った統合JSONを In_JSON（またはパスを JsonPath）へ入力する。
  2. モデルの通り芯交点をつなぐGH_GridIntersect の交点座標・ラベルを ModelGridPts / ModelGridLbls へ。図面座標だけで済ませたい場合はここを未接続でもよい。
  3. 範囲を調整するClipDepth / SecWidth / SecHeight などで、断面の切り取り範囲・奥行きを整える。
  4. 照合結果を確認するReport と CutPlanes / ViewDirs のプレビューで、各断面が正しい位置・向きに立っているか目で確かめる。
  5. Executorへ渡すPlan_JSON を GH_Make2DExecutor の PlanJSON へ。ProductKey は IFC属性検索 へつなぐと部材の絞り込みまで自動化できる。

5実データでの流れ

🧭 専用サンプルは準備中 ― 実案件の統合JSON＋モデルで動かします

プランナーは「2D図面から起こした統合JSON」と「3Dモデルの通り芯」が揃ってはじめて働くため、単体サンプルではなくパイプラインの中で試すのが分かりやすいです。

  1. 2D図面を用意サンプルモデル\02_2D図面.3dm のように通り芯記号(X1,X2,Y1…)・断面符号(A-A)入りの図面から、上流ノードで統合JSONを作る。
  2. 3Dモデルの交点を抽出サンプルモデル\01_柱梁フレーム.3dm で GH_GridIntersect を走らせ、通り芯交点の座標・名前を得る。
  3. プランナーで照合統合JSONと交点を本ノードへ渡してsolve。CutPlanes のプレビューで、図面の断面がモデルの正しい通りに立ったか確認する。
  4. 計画をExecutorへPlan_JSON を GH_Make2DExecutor へ流し、実際の断面線を起こす。

6注意点・既知の罠

  • ⚠ 交点名が図面とモデルで一致していないと照合できない

    プランナーは「名前」で図面とモデルを突き合わせます。図面側の通り芯符号（X1, Y2 など）と、モデル側の通り芯ラベルの表記が食い違っていると、その断面は位置が確定できません。上流で名前の付け方を揃えておくのが最重要ポイントです。

  • ⚠ 水平切断のZ基準は 1FL＝Z0 が前提

    水平切断（床伏図など）は切断高さ CutZ を計画に書き込みます。ここが1FL基準からずれると、下流の Executor で水平断面だけ線がゼロになる典型トラブルにつながります。Executor側のZオートフィットである程度は吸収されますが、まず計画のZ基準を確認してください。

  • ⚠ モデル交点なしの図面座標モードは配置基準が変わる

    ModelGridPts を未接続にすると、JSONの図面座標だけで計画を出せますが、断面の置かれる基準がモデル実座標ではなく図面座標になります。モデルと図面をぴったり重ねたいときは通り芯交点を接続してください。
