動作デモ

PREVIEW⟲ LOOP 10s

製品記号でブロックを検索して確保し、チェックリストで選んだ附帯パーツだけを本体から切り分ける——この一連の仕分けを10秒でループ再生しています。

1なにをするツール？

Make2Dパイプラインで「図面に描く部材」だけを取り出すためのノードです。モデルの中で ProductKey（製品記号）を部分一致で検索し、対象のブロック（＝ひとつの部材アセンブリ）を確保します。たとえば断面図に載せたい柱を、記号ひとつで一発で選び出せます。

さらに SplitPick のチェックリストで、確保したブロックの中身を本体（MainGeo）と附帯パーツ（SubGeo）に手で分けられます。柱本体は切断線、ガセットやベースプレートなどの附帯は見え線、というように Executor が描き分けるための素材になります。分類はブロック定義名をキーにして .3dm に記憶されるので、同じ部材なら次回も選択が復元されます。同じ形状を自動で附帯にまわす機能や、無人バッチ対応も備えています。

※ ProductKey／製品記号＝Teklaなどが部材に振る番号。ifc2rhino が焼き込んだ UserText 属性を検索対象にできる。／ 本体＝図面で切断線として描く主部材、附帯＝それに付くプレートやアングル類。

3使い方

  1. 検索語を入れるSearchKey に属性名、SearchValue に製品記号を入力する。プランナーの ProductKey を直結すると自動で入る。
  2. 実行して確保するRun を押すと、ヒットしたブロックがモデル上で白くハイライトされる。狙った部材が選べているか確認する。
  3. 本体と附帯を分けるSplitPick のチェックリストで、附帯にしたいパーツ（ガセット・ベースPL等）にチェックを入れる。チェック分が SubGeo、残りが MainGeo になる。
  4. 取りこぼしを手当てするリストに出ない一体化パーツは SubRefIds にIDを溜めると附帯へ寄せられる。飛び地の誤混入は FarTol で除外する。
  5. 下流へ渡すMainGeo / SubGeo を Executor へ、InstId を 接触検出 へつなぐ。

5サンプルモデルでデモ

🧩 サンプルモデル\01_柱梁フレーム.3dm

柱梁フレームにガセット／浮きピースが付いた素材。本体と附帯の仕分け、飛び地パーツの除外を試すのにちょうどよいです。

  1. モデルを開くサンプルモデル\01_柱梁フレーム.3dm を開く（またはワークセッションで参照）。
  2. 記号で検索SearchValue に柱の製品記号を入れて Run。対象ブロックが白くハイライトされる。
  3. 附帯を選ぶSplitPick でガセットや浮きピースにチェック。本体（柱）と附帯が色で分かれる。
  4. 飛び地を除外遠くに離れたパーツが混ざったら FarTol を絞って除外し、MainGeo / SubGeo がきれいに分かれることを確認する。

6注意点・既知の罠

  • ⚠ ブロック定義に飛び地パーツが混ざると附帯に誤混入する

    柱アセンブリの定義に、遠く離れた吊りピースなどの飛び地パーツが含まれていると、附帯として誤って拾ってしまいます。FarTol（遠隔パーツ除外）で、対角の最大アンカーから一定距離を超えるものを外してください。

  • ⚠ 検索は部分一致 ― 語がゆるいと余計なブロックまで拾う

    SearchValue は部分一致で検索します。記号が短すぎたり共通しすぎると、狙っていない部材まで一緒にヒットします。ハイライトを見て、拾いすぎていないか必ず確認してください。

  • ⚠ 属性が焼き込まれていないモデルでは記号検索できない

    検索対象は部材が持つ UserText 属性です。ifc2rhino で属性を焼き込んだモデルなら検索できますが、属性のない素のジオメトリでは記号でヒットしません。まず属性の有無を確認してください。

  • ⚠ 活発に更新中 ― 分類の自己修復に頼りきらない

    本ノードは頻繁に更新されており、モデルを切り替えても選択を保つ「型｜寸法の緩和照合による自己修復」を備えます。ただし定義名や寸法が大きく変わると再照合が要る場合があります。重要案件では仕分け結果を目視で確認してください。
