動作デモ

PREVIEW⟲ LOOP 10s

モデルに半透明の切断面を差し込み、その位置に断面線を起こしつつ矢視方向の背景だけを残す——統合JSONを介さず単体で完結する流れを10秒でループ再生しています。

1なにをするツール？

Make2Dパイプラインのお手軽ルートです。本流の プランナー は「2D図面から起こした統合JSON」を前提にしますが、このノードは統合JSONなしで、モデルと切断位置を渡すだけで断面をひとつ切れます。「今このモデルを、この線でスパッと切って断面図が欲しい」という単発の用途にぴったりです。

水平面（平面図・床伏図・天井伏図）でも鉛直面（断面図）でもカットでき、Make2D用のデータ（断面線・背景ジオメトリ・カメラ平面）と、Plan_JSON互換の CutInfo を単体で生成します。CutInfo はそのまま GH_Make2DExecutor の PlanJSON 端子へ直結できるので、「単体で切って、そのままExecutorで仕上げる」という短い経路が組めます。

※ 背景ジオメトリ＝切断面より奥に見える部材の輪郭。矢視方向にどちらを残すかで決まる。／ CutInfo＝切断計画をまとめたデータで、プランナーの Plan_JSON と互換。

3使い方

  1. 対象を渡す切りたいモデルを ID（Rhinoオブジェクト）または RefGeos（参照モデルのジオメトリ）で入力する。統合JSONは不要。
  2. 切り方を決めるMode で水平／鉛直を選び、CutPos（高さ）または CutLine（切断ライン）で位置を指定する。
  3. 向きを整える矢視が逆なら Flip で反転、切り取る奥行きは Depth で調整、断面には Name を付ける。
  4. 実行するPerformCut をオンにするとカットが走り、SectionCurves / BackgroundGeos が生成される。
  5. Executorへ直結するCutInfo を GH_Make2DExecutor の PlanJSON へつなげば、そのまま図面化・書き戻しまで進める。

5サンプルモデルでデモ

🧩 サンプルモデル\01_柱梁フレーム.3dm

2×1スパンの柱梁フレーム＋通り芯線＋ガセット／浮きピース入り。単体カットの練習にちょうどよい素材です。

  1. モデルを開くサンプルモデル\01_柱梁フレーム.3dm を開く。
  2. 対象を選ぶフレームを ID に渡し、Mode を鉛直、CutLine を柱間の1本に設定する。
  3. カットするPerformCut をオンにして、断面線と矢視方向の背景が出るのを確認する。
  4. Executorへ流すCutInfo を Executor へつなぎ、図面座標への配置まで通してみる（練習ではWriteBackはオフのまま）。

6注意点・既知の罠

  • ⚠ 用途を取り違えない ― 本格運用は Planner ルート

    PreCut は単発・単体運用向けの簡易ルートです。図面の統合JSONと連動させて多数の断面を体系的に起こすなら、本流の GH_Make2DPlanner → Executor を使ってください。役割を取り違えると二重管理になります。

  • ⚠ 見上げ（天井伏図）＋水平の直結は非対応

    見上げ方向の水平断面など、一部のモード組み合わせは直結に対応していません。想定外の向きで切りたいときは Flip や Mode の設定を見直し、結果のプレビューを必ず確認してください。

  • ⚠ PerformCut がオフだと何も生成されない

    カットは PerformCut がトリガです。設定を変えても出力が空のままなら、まずこのトリガがオンになっているかを確認してください。
